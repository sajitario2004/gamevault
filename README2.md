# gamevault2
implementaremos nuevos contenidos a gamevault1

# descripcion
ahora el boton implementado en gamevault de añadir juegos es funcional, este nos abrira un menu que nos permitira añadir un juego con contenidos como que calificacion y si lo hemos jugado, ahora otra cosa implementada es el boton de guardar que nos preguntara si estamos seguros del juego el cual vamos a implementar. Luego de añadir el juego se abrira un contenido emergente por debajo.

# funcionamiento
1. mover el contenido de nuestro
2. implementaremos el contenido en lib de nuestro proyecto
3. iniciaremos el main
4. pulsaremos el boton del lateral derecho en la esquina izquierda
